							MÓDULO 1 - CSS


style: Aplica CSS diretamente a um elemento HTML, permitindo definir estilos como cor, tamanho, margens, entre outros.

color: Define a cor do texto.

background-color: Define a cor de fundo do elemento.

font-size: Define o tamanho da fonte do texto.
font-family: Define a fonte a ser usada para o texto.
font-weight: Define a espessura da fonte (por exemplo, normal, bold).

text-align: Define o alinhamento horizontal do texto dentro do elemento.
text-decoration: Define decorações de texto, como sublinhado ou tachado.
text-transform: Define a transformação do texto, como maiúsculas ou minúsculas.
text-indent: Define a indentação da primeira linha de um bloco de texto. Você pode especificar o valor em unidades como pixels, porcentagem, etc.

line-height: Define a altura da linha para o texto.
letter-spacing: Define o espaçamento entre as letras do texto.

word-spacing: Define o espaçamento entre as palavras do texto.

margin: Define a margem ao redor do elemento.

padding: Define o preenchimento interno do elemento.
border: Define a borda ao redor do elemento.
border-radius: Define o raio das bordas arredondadas do elemento.

width: Define a largura do elemento.
height: Define a altura do elemento.

display: Define como o elemento é exibido (por exemplo, block, inline, flex).

overflow: Define o comportamento do conteúdo que excede o tamanho do elemento.

position: Define o método de posicionamento do elemento (por exemplo, static, relative, absolute, fixed).

top: Define a posição do elemento a partir do topo.
right: Define a posição do elemento a partir da direita.
bottom: Define a posição do elemento a partir do fundo.
left: Define a posição do elemento a partir da esquerda.

z-index: Define a ordem de empilhamento do elemento (sobreposição).

float: Define se o elemento deve flutuar à esquerda ou à direita.
clear: Define se o elemento deve limpar flutuações anteriores.

opacity: Define a opacidade do elemento (transparência).
visibility: Define a visibilidade do elemento (visível ou oculto).

box-shadow: Define uma sombra ao redor da caixa do elemento.
text-shadow: Define uma sombra para o texto.

transform: Aplica transformações 2D ou 3D ao elemento (como rotação ou escalamento).
transition: Define a transição suave entre estados de estilo.

cursor: Define o tipo de cursor exibido quando o mouse está sobre o elemento.

background-image: Define uma imagem de fundo para o elemento.
background-size: Define o tamanho da imagem de fundo.
background-repeat: Define se a imagem de fundo deve se repetir.
background-position: Define a posição da imagem de fundo.

transform-origin: Define o ponto de origem para transformações aplicadas ao elemento.

clip: Define uma região de recorte para o elemento.

filter: Aplica efeitos gráficos, como desfoque ou brilho, ao elemento.


justify-content: Define como o espaço disponível é distribuído entre os itens de um contêiner flexível (ou grid). Usado em contêineres com display flex ou grid para alinhar e distribuir itens ao longo do eixo principal.
Valores comuns:

flex-start: Alinha os itens ao início do contêiner.

center: Alinha os itens no centro do contêiner.

flex-end: Alinha os itens ao final do contêiner.

space-between: Distribui os itens uniformemente, com o primeiro item alinhado ao início e o último ao final do contêiner.

space-around: Distribui os itens uniformemente, com espaços iguais ao redor de cada item.

space-evenly: Distribui os itens uniformemente, com espaços iguais entre todos os itens, incluindo as bordas do contêiner.


font-size: Define o tamanho da fonte para o texto dentro de um elemento. Pode ser especificado em unidades absolutas (como pixels ou pontos) ou em unidades relativas (como em ou %).
Valores comuns:

px (pixels): Define o tamanho da fonte em pixels.
em: Define o tamanho da fonte relativo ao tamanho da fonte do elemento pai.
rem: Define o tamanho da fonte relativo ao tamanho da fonte da raiz (elemento <html>).
%: Define o tamanho da fonte relativo ao tamanho da fonte do elemento pai.
pt (pontos): Define o tamanho da fonte em pontos (1 pt = 1/72 polegada).


body: Refere-se ao elemento <body>de um documento HTML. Pode ser estilizado para definir a aparência geral da página, incluindo fontes, núcleos de fundo, margens e muito mais.
